Playbook

February 8, 2025

プレイブックを使って、Ansibleで実行するタスクを記述します。この例では、httpdをインストールするタスクをroleで定義したのでそれを呼ぶように記載します。

Playbookの作成

playbook.yml

- name: Deploy Apache Server
  hosts: all
  become: yes
  roles:
　  - httpd

このPlaybookは、alllホスト（Dockerコンテナ）に対してhttpdをインストールし、サービスを開始するタスクを実行します。
